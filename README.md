# Saffron & Sage - Restaurant Website

A warm, fully responsive multi-page website for a fictional cafe called **Saffron & Sage**. Built as a beginner web development project using only HTML and CSS, with no JavaScript or frameworks.

## Pages

- **Home** (index.html) - Hero section and welcome highlights
- **Menu** (menu.html) - Food and drinks with images and prices
- **About** (about.html) - Our story and values
- **Contact** (contact.html) - Contact details and a contact form

## Features

- Shared navbar and footer across all pages
- Warm restaurant color palette (brown, orange, cream)
- Responsive layout using Flexbox and CSS Grid
- Hover effects on cards and buttons
- Contact form UI

## What I Learned

- How to build a **multi-page website** and link pages together
- How to keep a **consistent navbar and footer** on every page
- How to use **Flexbox** and **CSS Grid** for layouts
- How to make a site **responsive** with media queries
- How to style **forms** and add **hover effects**
- How to host a website for free using **GitHub Pages**
